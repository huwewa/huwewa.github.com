把远程仓库克隆到本地

```
git clone git://github.com/username/xxx.git
```

在本地配置Git信息

```
git config --global user.name "username"
git config --global user.email "xxx@xxx.xxx"
```

在本地生成公匙

```
ssh-keygen -t rsa -C "xxx@xxx.xxx"
```

拷贝公匙并粘贴到github的账户设置里

```
pbcopy <~/.ssh/id_rsa.pub
```

（完）
